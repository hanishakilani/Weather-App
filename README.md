# 🌦️ Weather App

A modern and interactive weather application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using an external API and displays it in a clean, user-friendly interface.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Displays temperature, humidity, and weather condition  
- 🤒 Shows "feels like" temperature  
- 🎨 Dynamic background based on weather  
- ⏳ Loading indicator  
- ⌨️ Enter key support  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript  
- OpenWeather API  

---



## ⚙️ How to Run

1. Download or clone this repository  
2. Open the project folder  
3. Open `index.html` in your browser  

OR

Use Live Server in VS Code for best experience  

---
